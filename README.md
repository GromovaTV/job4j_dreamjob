DreamJob
=============

This project implements a platform for posting and searching for vacancies and resumes.

#### Dependencies

* Java
* Servlets, Filters
* JSP(JSTL), HTML, JavaScript(JQuery, Ajax), Bootstrap
* JDBC + PostgreSQL
* Liquibase
* Junit, Hamcrest, Mockito, H2
* Maven
* Tomcat
* Log4j, Slf4j
* Checkstyle
